Grid

The grid system is designed to provide a fluid grid that is very simple and highly semantic.

Wrap rows with a row div then simply use the column wrapper to specify the number of columns you want col-2. Every direct child within the column wrapper will be given a float, made a block, and given the appropiate percentage width. 

<div class="row">
  <div class="col-2">
    <div> ... </div>
    <div> ... </div>
  </div>
</div>

<div class="row">
  <div class="col-3">
    <div> ... </div>
    <div> ... </div>
    <div> ... </div>
  </div>
</div>
